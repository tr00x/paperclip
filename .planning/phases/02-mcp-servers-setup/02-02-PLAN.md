Pause execution and ask user to complete the following manual steps that cannot be automated (require Telegram app and Google Account web UI):

**Telegram Bot Setup (5 minutes):**
1. Open Telegram app, search for @BotFather
2. Send `/newbot`, follow prompts to name it (e.g., "AmriTech AI Bot")
3. Copy the bot token (format: `123456:ABC-DEF1234ghIkl-zyx57W2v1u123ew11`)
4. Send `/mybots` -> select bot -> Bot Settings -> Group Privacy -> Turn off
5. Create a new group chat in Telegram, add the bot + team members (@Berik, @Ula, @Timur)
6. Send any message in the group (e.g., `/start`)
7. In a browser, visit: `https://api.telegram.org/bot<YOUR_TOKEN>/getUpdates`
8. Find the `"chat":{"id":-XXXXXXXXX}` value -- this is the group chat_id (negative number)

**Gmail App Password (3 minutes):**
1. Go to https://myaccount.google.com/security
2. Ensure 2-Step Verification is ON
3. Go to App Passwords (search "App Passwords" in Google Account settings)
4. Select "Mail", generate password
5. Copy the 16-character password (format: `xxxx xxxx xxxx xxxx`)

**Request these 4 values from user:**
- TELEGRAM_BOT_TOKEN
- TELEGRAM_CHAT_ID
- MCP_EMAIL_ADDRESS
- MCP_EMAIL_PASSWORD

Using the credentials provided by the user in Task 1:

1. Create `mcp-servers/telegram/.env`:
   ```
   TELEGRAM_BOT_TOKEN=<user-provided token>
   TELEGRAM_CHAT_ID=<user-provided chat_id>
   ```

2. Register Telegram MCP in Claude Code project scope:
   ```
   claude mcp add telegram \
     -e TELEGRAM_BOT_TOKEN=<token> \
     -e TELEGRAM_CHAT_ID=<chat_id> \
     -- npx @parthj/telegram-notify-mcp
   ```

3. Verify Telegram bot by sending a test message via curl:
   ```
   curl -s -X POST "https://api.telegram.org/bot<TOKEN>/sendMessage" \
     -d "chat_id=<CHAT_ID>&text=AmriTech AI HQ connected! MCP setup complete."
   ```
   Expected: JSON response with `"ok":true`.

4. Create `mcp-servers/email/.env`:
   ```
   MCP_EMAIL_ADDRESS=<user-provided address>
   MCP_EMAIL_PASSWORD=<user-provided app password>
   MCP_EMAIL_IMAP_HOST=imap.gmail.com
   MCP_EMAIL_IMAP_PORT=993
   MCP_EMAIL_SMTP_HOST=smtp.gmail.com
   MCP_EMAIL_SMTP_PORT=465
   ```

5. Register Email MCP in Claude Code project scope:
   ```
   claude mcp add email \
     -e MCP_EMAIL_ADDRESS=<address> \
     -e MCP_EMAIL_PASSWORD=<app-password> \
     -e MCP_EMAIL_IMAP_HOST=imap.gmail.com \
     -e MCP_EMAIL_IMAP_PORT=993 \
     -e MCP_EMAIL_SMTP_HOST=smtp.gmail.com \
     -e MCP_EMAIL_SMTP_PORT=465 \
     -- npx @codefuturist/email-mcp stdio
   ```

6. Update `mcp-servers/README.md`:
   - Mark telegram as "Configured"
   - Mark email as "Configured"
   - All 5 servers should now show "Configured"
   - Add final verification section listing all 5 servers

7. Ensure `mcp-servers/*/.env` lines are in `.gitignore`.
