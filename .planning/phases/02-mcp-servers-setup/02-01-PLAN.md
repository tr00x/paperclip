1. Install pandoc via Homebrew:
   ```
   brew install pandoc
   ```
   Verify: `pandoc --version` returns a version number.

2. Create mcp-servers/ directory structure:
   ```
   mkdir -p mcp-servers/twenty-crm mcp-servers/telegram mcp-servers/email mcp-servers/docs
   ```

3. Create `mcp-servers/twenty-crm/.env` with these exact values:
   ```
   TWENTY_API_KEY=<copy exact value from twenty-crm/.env>
   TWENTY_BASE_URL=http://localhost:5555
   ```
   CRITICAL: Copy the TWENTY_API_KEY value exactly from twenty-crm/.env. Do NOT invent a value.

4. Install twenty-mcp-server globally:
   ```
   npm install -g twenty-mcp-server
   ```

5. Register Twenty CRM MCP in Claude Code project scope:
   ```
   claude mcp add twenty-crm \
     -e TWENTY_API_KEY=<exact key from twenty-crm/.env> \
     -e TWENTY_BASE_URL=http://localhost:5555 \
     -- npx twenty-mcp-server start
   ```

6. Create `mcp-servers/README.md` with overview of all MCP servers for this project:
   - Title: "AmriTech AI HQ - MCP Servers"
   - Table of servers: name, package, purpose, status
   - Start commands for each server
   - Mark Twenty CRM as "Configured", others as "Pending"

7. Add `mcp-servers/*/.env` to .gitignore if not already present (contains secrets).
  </action>

    <automated>pandoc --version && npm list -g twenty-mcp-server && cat mcp-servers/twenty-crm/.env | grep TWENTY_BASE_URL && claude mcp list 2>/dev/null | grep twenty-crm</automated>
  </verify>

    - `mcp-servers/README.md` exists with at least 20 lines
