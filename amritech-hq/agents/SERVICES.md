## Стратегия продажи

```
IT pain (крючок) → Managed IT ($3-5k/мес)
    ↓ через 1-2 месяца
Увидели больше проблем → Проект ($5-50k)
    ↓ через 3-6 месяцев
Доверие выросло → AI/Автоматизация ($5-50k/мес)
    ↓
Tech partner на всё → $10-20k/мес + проекты
```

**Один клиент в полном потенциале: $10-20k MRR + $50-100k проектов в год.**
