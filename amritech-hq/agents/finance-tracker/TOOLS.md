# Finance Tracker — Tools

## Twenty CRM MCP
- Запрос инвойсов по статусу (sent, paid, overdue)
- Обновление статуса инвойса
- Запрос активных контрактов для MRR расчёта
- Логирование оплат

## Paperclip API
- Получение [INVOICE] задач
- Создание sub-tasks SDR (напоминания)
- Эскалация CEO (urgent comments)
- Создание [REPORT] задач (еженедельный отчёт)
- Комментарии к задачам
