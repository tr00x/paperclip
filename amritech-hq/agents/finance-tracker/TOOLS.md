# Finance Tracker — Tools

## Twenty CRM — `search_invoices`, `create_invoice`, `update_invoice`, `get_client`

| Tool | When |
|------|------|
| `search_invoices` | Найти инвойсы по имени |
| `create_invoice` | Создать новый инвойс |
| `update_invoice` | Обновить статус (sent → paid / overdue) |
| `get_client` | Получить данные клиента для MRR |

## Paperclip API
- Получение [INVOICE] задач
- Создание sub-tasks SDR (напоминания)
- Эскалация CEO (urgent comments)
- Создание [REPORT] задач (еженедельный отчёт)
- Комментарии к задачам
