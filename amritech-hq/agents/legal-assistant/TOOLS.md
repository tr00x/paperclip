# Legal Assistant — Tools

## Web Search (встроенный Claude)
- Проверка актуальных требований NY/NJ law
- SAM.gov compliance requirements
- Federal acquisition regulations (FAR/DFARS)
- Insurance and bonding requirements по jurisdiction
- Set-aside eligibility criteria

## Docs MCP (Office-Word-MCP + mcp-pandoc)
- Генерация MSA, NDA, SLA в формате DOCX
- Конвертация в PDF через pandoc
- Форматирование: professional layout, headers, footers

## Paperclip API
- Получение задач от CEO (compliance checks, document generation, contract review)
- Чтение контекста из родительских задач ([TENDER], [LEAD])
- Публикация вердиктов и документов как комментарии
- Обновление статуса задач
