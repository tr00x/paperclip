# CRM (Twenty) — Как работать

## Что это

Twenty CRM — наша база данных. Здесь хранится ВСЁ: лиды, клиенты, контракты, оплаты. Все агенты читают и пишут сюда. Если данных нет в CRM — агенты слепые.

## Как зайти

- **Сейчас:** `http://localhost:5555` (только с компа Tim'а)
- **Скоро:** `https://crm.yourcompany.example.com` (из любого браузера, из дома)

## Основные объекты

### Leads (Лиды)
Потенциальные клиенты которых нашёл Hunter.

**Что видишь:**
- Название компании
- Контактное лицо (Decision Maker) + email + телефон
- ICP Score (0-100) — чем выше, тем лучше
- Industry (ниша)
- Location
- Signals (почему этой компании нужен IT)
- Status (где лид в воронке)

**Статусы лида:**

| Статус | Что значит | Кто отвечает |
|--------|-----------|-------------|
| `new` | Только создан Hunter'ом | Ждёт SDR |
| `qualified` | Есть email, ICP 60+ | SDR берёт в работу |
| `contacted` | Email отправлен | SDR ждёт ответ |
| `engaged` | Лид ответил | Alex решает |
| `meeting_set` | Звонок назначен | Closer готовит briefing |
| `closed_won` | Стал клиентом! 🎉 | Onboarding |
| `closed_lost` | Отказ | Архив |
| `nurture` | Не сейчас, вернёмся через 30 дней | Hunter пересканирует |

### Clients (Клиенты)
Активные клиенты с контрактами.

### Invoices (Инвойсы)
Счета и оплаты.

### Tenders (Тендеры)
Госзакупки и RFP.

## Что записывать в CRM

### Для Alex:
- [ ] Внести ВСЕХ текущих клиентов (название, контракт, MRR, дата renewal)
- [ ] После каждого звонка/встречи — обновить Notes
- [ ] Когда одобрил ответ на лида — записать решение

### Для Sam:
- [ ] После КАЖДОГО звонка клиенту:

| Поле | Что писать | Пример |
|------|-----------|--------|
| Кому звонил | Имя + компания | "John Smith, ABC Corp" |
| О чём | Тема разговора | "Renewal контракта" |
| Результат | Что решили | "Продлевают, подпишут на след. неделе" |
| Next step | Что дальше | "Отправить контракт в среду" |

### Где записывать:
1. Открой CRM → Leads (или Clients)
2. Найди компанию
3. Открой карточку
4. Notes → добавь запись с датой

### Формат записи:
```
[2026-03-22] Звонил John Smith (ABC Corp)
Тема: renewal контракта
Результат: продлевают, довольны сервисом
Next: отправить обновлённый контракт к среде
— Sam
```

## Зачем это НАСТОЛЬКО важно

Без данных в CRM:

1. **SDR отправит cold email действующему клиенту** — позор, клиент обидится
2. **Closer не знает контекст** — Alex идёт на звонок неподготовленный
3. **Contract Manager не видит renewal** — контракт истечёт и никто не заметит
4. **Finance не знает MRR** — не можем считать доход
5. **CEO не видит pipeline** — принимает плохие решения

**Одна запись в CRM = 5 минут. Потеря клиента = $36,000/год.**

## Views в CRM (полезные фильтры)

| View | Что показывает |
|------|---------------|
| Full Pipeline | Все лиды |
| Outreach Pipeline | Лиды в outreach (email_sent, follow_up) |
| Hot Leads (70+) | Лиды с высоким ICP score |
| Missing Email | Лиды без email DM (нужен enrichment) |
| By Industry | Лиды по нишам |
| By MRR Potential | Лиды по потенциальному MRR |
